    Introduction

    Welcome to the WebStreak Inventory API! You can use our API to access inventory API endpoints, which can get information on various dealers, vehicles, and leads in our database.

    WebStreak API access is restricted to our partner companies.

    We have language bindings in Shell and Ruby. You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    Production API is available at api.webstreak.com .

    Dealers

    Get All Dealers

    The above command returns JSON structured like this:

    This endpoint retrieves all dealers.

    HTTP Request

    GET http://api.webstreak.com/dealers

    Get a Specific Dealer

    The above command returns JSON structured like this:

    This endpoint retrieves a specific dealer.

    HTTP Request

    GET http://api.webstreak.com/dealers/<ID>

    URL Parameters

    Parameter Description
    ID The ID of the dealer to retrieve

    Vehicles

    Get All Vehicles

    This endpoint retrieves all vehicles.

    Requests for all vehicles with no query parameters may be extremely slow.

    The above command returns JSON structured like this:

    HTTP Request

    GET http://api.webstreak.com/vehicles

    URL Parameters

    Parameter Description
    dealer_id query vehicles for a specific dealer
    vin query by vin number
    stock query by stock number
    updated_since all vehicles updated/created after specified time
