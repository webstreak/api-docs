    Introduction

    Welcome to the WebStreak Inventory API! You can use our API to access inventory API endpoints, which can get information on various dealers and vehicles in our database.

    WebStreak API access is restricted to our partner companies.

    You can view code examples in the dark area to the right.

    Production API is available at websites.webstreak.com/api/public .

    Authentication

    Example call to dealers endpoint with api key:

    curl http://websites.webstreak.com/api/public/dealers?api_key=6b248ea92c8fa7cf7428cbae62e1ede1

    Authentication is accomplished by providing your api key with each request. API access is limited to our partner companies.

    Get a Specific Dealer

    curl http://websites.webstreak.com/api/public/dealers/15

    The above command returns JSON structured like this:

    {
"id": 15,

    HTTP Request

    GET http://websites.webstreak.com/api/public/dealers/<ID>

    URL Parameters

    Parameter Description
    ID The ID of the dealer to retrieve
